# eu4-font-gothic
EU4日本語化MOD用のゴシックフォントです。

## 概要
日本語化MODのフォントをゴシック体に変更します。

おまけで画面右端の一覧表および他国からの通知欄の横幅を広げます。

## 互換性
鉄人モードで使用できます（Steamの実績を獲得できます）

インターフェイスを変更する一部のMODとは同時に使用できません。

## ご注意
他国からの通知は国旗の配置がEU4本来の順序とは逆になっています。
